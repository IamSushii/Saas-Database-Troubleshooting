Project: SaaS Billing & API Troubleshooting Database
Role: Technical Solutions Engineer Simulation

What I Built:
I engineered a local relational database (MySQL) to simulate the backend of a SaaS platform. I injected raw data representing users, financial transactions, and server logs.

The Problem I Solved:
I utilized INNER JOIN and aggregate functions (GROUP BY) to successfully trace a surge of failed Premium subscription payments back to a recurring 500 Internal Server Error on the /checkout API endpoint.
